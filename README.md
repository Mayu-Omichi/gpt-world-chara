# gpt-world-chara
GPTでキャラの物語(バックストーリー)を生成し、それをもとにTerminal上でそのキャラクターと対話することができるプログラムです。

自分のメモ用に置いているため、実行方法は後日記載予定です

.env_sampleは.envのサンプルです。
.envファイルを別で作成し、ディレクトリ情報やAPI情報などを記入し、gitignoreファイルに.envを除外するように書き込んでください。
